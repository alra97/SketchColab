SketchColab Help

Overview

Welcome to SketchColab! This application allows you to create and join drawing rooms where you can sketch and chat with other players. Here’s a guide to help you navigate and use the features effectively.

Lobby

The lobby is where you can create or join a drawing room:

  • Name: Enter your name here.
  • Room: Enter a name for the room you want to create or join.
  • Create Room: Click this button to create a new drawing room with the specified name.
  • Join Room: Click this button to join an existing room by its name.

Drawing Canvas

The drawing canvas is where you will do your sketching. You can use the following tools:

  • Color Picker: Select a color to draw with.
  • Clear Your Canvas: Click this button to clear the entire canvas.
  • Eraser: Switch between drawing and erasing. The button toggles between 'Brush' and 'Eraser' mode.

Chat

Interact with other players using the chat feature:

  • Chat Input: Type your messages here.
  • Send Button: Click this button to send your message to the chat.
